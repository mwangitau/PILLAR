# **App Name**: Pillar

## Core Features:

- AI-Powered Personalized Plan Generation: The app's core feature involves a sophisticated AI engine utilizing Google Gemini to generate personalized manual based on user input, collected answers, user profile and a selection of prompt templates, offering tailored advice on identity, daily routines, habits, sobriety and accountability, and stores immutable outputs to avoid repeat calls and for auditing, using a strict JSON schema to define each area.
- Interactive AI Onboarding Chat: An engaging chat UI that guides users through a series of onboarding questions, leveraging AI-driven question-and-answer logic to progressively build a detailed profile and understand the user's specific needs and goals.
- Comprehensive Habit and Routine Management: Enable users to create, manage, and track their habits and daily routines. Include scheduling, logging, streak tracking, and flexible time-based reminders to promote consistent adherence.
- AI-Driven Journaling with Mood and Sentiment Analysis: Provide a digital journaling feature, allowing users to record their thoughts, feelings, and experiences. Employ sentiment analysis tool to automatically detect and tag mood patterns for insightful self-reflection.
- Secure Accountability and Sharing System: Implement a secure accountability system that enables users to invite partners, define permission levels, and share weekly reports. Log consent and track user interactions, guaranteeing privacy while maintaining responsible sharing.
- Versatile Finance Tracker with Tithe Handling: Develop a light ledger within the app that simplifies tracking income, expenses, tithes and financial categories. Provide customizable reports and facilitate M-Pesa integration for easy premium subscriptions and tithe payments within the Kenyan userbase.
- Exportable Report Generation: Allows users to generate shareable PDF versions of manuals, accountability reports, journals, or finance summaries on request using background workers that compile user data, generate AI summaries if needed, and save them on S3 to be accessed by the frontend.

## Style Guidelines:

- Primary color: A warm coral (#FF7F50) to represent vitality and growth, balancing energy and serenity.
- Background color: A desaturated light peach (#F5DEB3) to create a cozy and inviting atmosphere, reminiscent of gentle mornings and fresh starts.
- Accent color: A soft lavender (#E6E6FA) to add a touch of tranquility and sophistication, encouraging peaceful reflection and balanced emotional responses.
- Body font: 'PT Sans', a versatile sans-serif for comfortable readability of the app's text and article-style content.
- Headline font: 'Playfair', a modern serif font with high contrast lines to convey elegance.
- Utilize clean and minimalist icons for key actions and categories, focusing on intuitive recognition and seamless integration within the UI.
- Implement a card-based layout with rounded corners and subtle shadows for a modern and organized feel, ensuring easy navigation and content prioritization.
- Incorporate subtle animations and transitions to enhance user engagement and provide feedback, such as smooth loading indicators and playful micro-interactions.
